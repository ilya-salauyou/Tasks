Использование Сложных Типов(Перечислений)

Требования:
1. Входные поля: Имя(FirstName), Фамилия(LastName),
	Тип Клиента(ClientType) (вводится как строка: "Премиум", "Стандарт", "Новый").
2. Форматирование Имени/Фамилии: Стандартный регистр.
3. Структурирование данных: Создать C# enum (перечисление)
	для возможных типов клиентов (Premium, Standard, New).
4. Валидация Типа: Модуль должен попытаться преобразовать
	входную строку Тип Клиента в созданное перечисление.
	Если преобразование не удалось (введена неверная строка),
	по умолчанию присвоить тип "Стандарт".
5. Вывод: Отформатированное полное имя и числовое/текстовое
	значение типа клиента, полученное из перечисления.
